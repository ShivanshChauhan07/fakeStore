About This Project

This is a simple e-commerce demo application built to learn Angular, routing, component communication, and API integration.

Purpose

The goal of this project is to practice real-world frontend concepts like product listing, search, sorting, routing, and form handling.

Tech Stack

  • Angular (Standalone Components)
  • Tailwind CSS
  • FakeStore API
  • TypeScript

Features

  • Product listing
  • Search and sort
  • Add new product
  • Client-side routing
